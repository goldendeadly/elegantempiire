# Elegant Empiire

A premium editorial comparison platform for luxury fashion brands, designed with an "Architectural Silence" aesthetic.

## Overview

This project is a high-performance React application built with Vite and Tailwind CSS. It features a scalable comparison engine that allows for in-depth analysis of luxury fashion brands (e.g., The Row vs. Lemaire).

## Key Features

- **Premium Aesthetic:** Custom "Winter Fog & Stone" color palette with Playfair Display typography.

- **Scalable Architecture:** Centralized data management for easy addition of new brand comparisons.

- **Performance:** Optimized for speed with lightweight assets and efficient rendering.

- **Responsive Design:** Flawless experience across desktop, tablet, and mobile devices.

## Tech Stack

- **Framework:** React 19

- **Build Tool:** Vite

- **Styling:** Tailwind CSS 4

- **Package Manager:** pnpm

## Deployment

This project is optimized for deployment on Vercel.

1. Connect your GitHub repository to Vercel.

1. Vercel will automatically detect the Vite configuration.

1. Deploy!
